# 網頁動畫展演與技術解構網站

這是一個展示各種 Web 動畫效果（如 Hover 互動、進場動畫、捲動視差、3D 渲染等），並能自動分析與解構該動畫背後所使用之技術棧與原始碼的網站。整體視覺採用乾淨的卡片式網格佈局。

## 系統需求
- Node.js (建議版本 18 以上)
- npm 或 pnpm

## 啟動與安裝指南

1. **安裝相依套件**
   ```bash
   npm install
   ```

2. **環境變數設定**
   請複製 `.env.example` 檔案並重新命名為 `.env.local`，然後填入所需的 API 金鑰（例如 Gemini API Key 等）。

3. **啟動本地端開發伺服器**
   ```bash
   npm run dev
   ```
   伺服器啟動後，請在瀏覽器前往 [http://localhost:3000](http://localhost:3000) 即可瀏覽網站。
